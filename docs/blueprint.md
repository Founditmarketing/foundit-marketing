# **App Name**: Revenue Driver

## Core Features:

- Performance Data Display: Visualize key performance indicators (KPIs) such as leads generated, client revenue won, and other relevant metrics, in a dashboard format. Track live heatmap conversion data to visualize user intent.
- AI-Powered Ad Bidding: Employ a tool for AI-driven bidding strategies for Google Ads based on real-time data analysis to optimize cost-per-acquisition.
- Client Retention Tracking: Display client retention rate. Allows filtering to understand why clients churn and predict which clients may leave

## Style Guidelines:

- Primary color: Vibrant orange (#FF7000) to represent energy and action.
- Background color: Desaturated orange (#FAF0E6) to provide a gentle backdrop that doesn't distract from the content.
- Accent color: Soft red (#F06030), is used to give high-emphasis feedback such as for errors, or things requiring review.
- Body and headline font: 'Inter' (sans-serif) for a clean, authoritative feel.
- Use sharp, modern icons that complement the tech-focused aesthetic.
- Employ a grid-based layout inspired by the 'golden ratio' to present information in a visually harmonious and conversion-focused manner. Follow a 12-column grid system with clear, actionable sections.
- Subtle animations, like progress bars, enhance user experience without being distracting. For AI bidding status, implement a subtle glow effect to emphasize its operation.
